# 🎉 Демонстрация проекта PizzaOrder

## ✅ Что запущено и работает:

### 📱 **Frontend (React Native + Expo)**

Приложение запущено и готово к просмотру!

**Основные функции:**
- ✅ Главная страница со списком пицц (4 тестовые пиццы)
- ✅ Поиск пицц по названию, описанию и ингредиентам
- ✅ Детальная страница пиццы с возможностью выбора количества
- ✅ Корзина с добавлением/удалением товаров
- ✅ Сохранение корзины в AsyncStorage (данные сохраняются между сессиями)
- ✅ Экран оформления заказа
- ✅ Экран авторизации/регистрации
- ✅ Админ-панель (доступна для пользователей с ролью ADMIN)

**Интерфейс:**
- 🎨 Современный UI на React Native Paper
- 📱 Адаптивный дизайн для всех платформ
- 🔍 Поиск в реальном времени
- 🛒 Счетчик товаров в корзине
- 👤 Управление авторизацией

### 🔧 **Backend (NestJS)**

Backend скомпилирован и готов к запуску!

**Реализованные модули:**
- ✅ Prisma ORM с полной схемой базы данных
- ✅ JWT аутентификация
- ✅ Модуль пицц (CRUD)
- ✅ Модуль заказов
- ✅ Админ-панель API
- ✅ Валидация данных
- ✅ Guards для защиты роутов
- ✅ CORS настроен для frontend

**API Endpoints:**
```
POST   /api/auth/register      - Регистрация
POST   /api/auth/login         - Вход
GET    /api/pizzas             - Список пицц
GET    /api/pizzas/:id         - Детали пиццы
POST   /api/orders             - Создать заказ
GET    /api/orders             - Мои заказы
GET    /api/admin/orders        - Все заказы (ADMIN)
PATCH  /api/admin/orders/:id   - Изменить статус (ADMIN)
GET    /api/admin/pizzas        - Список пицц (ADMIN)
POST   /api/admin/pizzas        - Создать пиццу (ADMIN)
PATCH  /api/admin/pizzas/:id   - Обновить пиццу (ADMIN)
DELETE /api/admin/pizzas/:id   - Удалить пиццу (ADMIN)
```

## 🎯 Как использовать прямо сейчас:

### 1. **Просмотр интерфейса (Frontend работает без backend)**

1. Откройте терминал Expo (должен быть запущен)
2. Нажмите `w` для Web версии
3. Или отсканируйте QR-код мобильным приложением Expo Go

**Что вы увидите:**
- Главную страницу с 4 пиццами
- Возможность поиска
- Кликабельные карточки пицц
- Кнопку добавления в корзину
- Иконку корзины с счетчиком

### 2. **Тестирование функций:**

**Корзина:**
- Добавьте пиццу в корзину
- Откройте корзину (иконка в шапке)
- Измените количество
- Удалите товар
- Данные сохраняются локально!

**Детали пиццы:**
- Кликните на любую карточку пиццы
- Увидите детальное описание
- Можете выбрать количество
- Добавить в корзину с нужным количеством

**Поиск:**
- Введите в поисковую строку: "пепперони", "сыр", "томаты"
- Список фильтруется в реальном времени

## ⚙️ Для полной функциональности:

**Требуется настроить PostgreSQL:**

1. Установите PostgreSQL
2. Создайте `.env` в `backend/`:
   ```env
   DATABASE_URL="postgresql://user:password@localhost:5432/pizzaorder?schema=public"
   JWT_SECRET=your-secret-key-here
   JWT_EXPIRES_IN=7d
   PORT=3000
   ```

3. Выполните миграции:
   ```bash
   cd backend
   npx prisma migrate dev --name init
   npx prisma generate
   ```

4. Перезапустите backend
