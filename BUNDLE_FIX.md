## Если bundle все еще не загружается:

### 1. Перезапустите с очисткой кэша

```bash
# Остановите текущий процесс (Ctrl+C)
# Затем:
npx expo start --clear
```

### 2. Проверьте консоль браузера

Откройте DevTools (F12) и проверьте:
- Есть ли ошибки в консоли
- Какой статус возвращает bundle URL

### 3. Попробуйте другой порт

Если порт 8081 занят, Expo использует 8082. Это нормально.

### 4. Проверьте терминал Expo

В терминале, где запущен `expo start`, должны быть сообщения о:
- Компиляции bundle
- Ошибках, если они есть
